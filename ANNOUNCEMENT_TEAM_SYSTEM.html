Sistema de Anuncios de Equipo

📢 ¿Qué es?

Hemos implementado un sistema completo de anuncios que permite a los administradores de equipo comunicar información importante a todos los miembros de forma visual y efectiva.

✨ Características Principales

  • ✓
    Editor WYSIWYG con Modo HTML

    Crea anuncios con formato rico usando el editor visual o edita directamente el código HTML.

  • ✓
    Estilos Personalizados

    Usa todas las clases de TailwindCSS para crear anuncios atractivos y profesionales.

  • ✓
    Fechas Opcionales

    Define fechas de inicio y fin para anuncios temporales que se ocultan automáticamente.

  • ✓
    Control de Activación

    Activa o desactiva anuncios sin eliminarlos, manteniendo un historial completo.

  • ✓
    Visualización Inteligente

    Los anuncios aparecen colapsados por defecto en la página de eventos para no ser intrusivos.

👨‍💼 Para Administradores

📍 Ubicación: Ve a Configuración del Equipo → pestaña Anuncios

➕ Crear anuncio: Haz clic en "Crear Anuncio" y usa el editor para dar formato a tu mensaje.

✏️ Editar: Puedes editar cualquier anuncio existente haciendo clic en el ícono de lápiz.

🔄 Activar/Desactivar: Usa el botón de encendido para mostrar u ocultar anuncios temporalmente.

🗑️ Eliminar: Elimina anuncios que ya no sean necesarios (requiere confirmación).

👥 Para Usuarios

Los anuncios activos de tu equipo aparecen en la página de Eventos, en un panel azul con el ícono de megáfono 📢.

Haz clic en el panel para expandirlo y ver todos los anuncios disponibles con su información completa.

Los anuncios pueden incluir fechas de vigencia y muestran quién los publicó y cuándo.

🔒 Seguridad

Esta funcionalidad ha sido desarrollada con máxima prioridad en seguridad:

  • ✓ Autorización basada en Policies: Solo administradores de equipo pueden gestionar anuncios
  • ✓ Protección IDOR: Verificación estricta de permisos en cada operación
  • ✓ Sanitización HTML: Todo el contenido HTML es filtrado para prevenir ataques XSS
  • ✓ Mass Assignment Protection: Campos sensibles protegidos contra modificación
  • ✓ Auditoría completa: Registro de quién creó cada anuncio y cuándo

🚀 Detalles Técnicos

Frontend

  • • Editor Quill.js 1.3.7 (auto-alojado)
  • • Livewire 2.x para reactividad
  • • Alpine.js para UI interactiva
  • • TailwindCSS para estilos

Backend

  • • Laravel Policies para autorización
  • • HTMLPurifier 4.19 para sanitización
  • • Migraciones con FK cascade
  • • Eloquent con scopes y relaciones
