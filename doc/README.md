# CTH Application Documentation

This directory contains comprehensive documentation for the CTH (Control de Tiempo y Horarios) application.

## Language Structure

The documentation is organized by language in the following directories:

- **`es/`** - Spanish documentation (Documentación en español)
- **`en/`** - English documentation

## Available Documentation

### Core Documentation
- **Installation Guide** - Setup and deployment instructions
- **User Manual** - Complete user guide for all application features
- **Administrator Guide** - Administrative functions and team management
- **API Documentation** - Technical API reference
- **Security Guide** - Security policies and best practices

### Technical Documentation
- **Database Schemas** - Database structure and migrations
- **Command Reference** - Artisan commands and utilities
- **Deployment Guide** - Production deployment procedures
- **Architecture Overview** - System architecture and design patterns

### Release Documentation
- **Changelog** - Version history and feature updates
- **Migration Guide** - Upgrade procedures between versions
- **Security Advisories** - Security updates and fixes

## Contributing to Documentation

When adding new documentation:

1. Create the document in Spanish first (`es/` directory)
2. Translate to English (`en/` directory)
3. Ensure both versions are synchronized
4. Update this README with the new document reference

## Document Naming Convention

Use descriptive, lowercase filenames with hyphens:
- `installation-guide.md`
- `user-manual.md`
- `api-reference.md`
- `security-policies.md`
