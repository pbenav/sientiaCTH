### Dashboard Principal
- `dashboard-principal.png` - Vista general del dashboard principal
- `metricas-dashboard.png` - Tarjetas con las métricas principales
- `grafico-horas.png` - Gráfico mostrando evolución de horas trabajadas

### Sistema de Fichaje
- `smart-clockin.png` - Interfaz del Smart Clock-In
- `evento-manual.png` - Formulario para crear evento manual
- `fichaje-excepcional.png` - Diálogo de confirmación para fichaje excepcional

### Gestión de Eventos
- `formulario-evento.png` - Formulario completo para crear un evento

- `edicion-masiva.png` - Interfaz para edición masiva de eventos

### Calendar y Horarios

- `configuracion-horarios.png` - Panel de configuración de horarios de trabajo

### Informes y Estadísticas

## 📸 Instrucciones para Capturas

### Formato Recomendado
- **Formato**: PNG (mejor calidad para interfaces)
- **Resolución**: Mínimo 1200px de ancho
- **Calidad**: Alta definición para legibilidad

### Consideraciones de Privacidad
- **Datos sensibles**: Usar datos de prueba o difuminar información personal
- **Usuarios**: Crear usuarios de ejemplo para las capturas
- **Fechas**: Usar fechas coherentes y actuales

### Estilo Visual
- **Consistencia**: Mismo navegador y configuración visual
- **Recortes**: Incluir contexto suficiente pero sin elementos innecesarios
- **Tamaño**: Optimizar para web (100-200KB por imagen)

## 🔄 Proceso de Actualización

1. Tomar las capturas de pantalla según la lista
2. Optimizar las imágenes para web
3. Guardar en este directorio con los nombres exactos
4. Verificar que se muestran correctamente en el manual
5. Hacer commit y push de las nuevas imágenes
