## 📸 Screenshot Instructions

### Recommended Format
- **Format**: PNG (better quality for interfaces)
- **Resolution**: Minimum 1200px width
- **Quality**: High definition for readability

### Privacy Considerations
- **Sensitive data**: Use test data or blur personal information
- **Users**: Create example users for screenshots
- **Dates**: Use coherent and current dates

### Visual Style
- **Consistency**: Same browser and visual configuration
- **Cropping**: Include sufficient context without unnecessary elements
- **Size**: Optimize for web (100-200KB per image)

## 🔄 Update Process

1. Take screenshots according to the list
2. Optimize images for web
3. Save in this directory with exact names
4. Verify they display correctly in the manual
5. Commit and push the new images
