# Manual de Usuario

Este documento describe cómo instalar, configurar y usar la aplicación.

## Instalación

1. Clonar el repositorio.
2. Ejecutar `composer install`.
3. Configurar el archivo `.env`.
4. Ejecutar `php artisan migrate`.
5. Iniciar el servidor con `php artisan serve`.

## Uso

- Navegar a la URL principal.
- Utilizar los menús para generar reportes, etc.
