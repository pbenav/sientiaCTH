# Manuel d'Utilisateur

Ce document décrit l'installation, la configuration et l'utilisation de l'application.

## Installation
1. Cloner le dépôt.
2. Exécuter `composer install`.
3. Copier `.env.example` en `.env` et configurer la base de données.
4. Exécuter `php artisan migrate`.
5. Démarrer le serveur de développement avec `php artisan serve`.

## Utilisation
- Naviguer vers la page d'accueil.
- Utiliser le menu pour générer des rapports, voir le calendrier, etc.
